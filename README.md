## 目录结构

```
    ├─src  	            （项目核心目录）
    |  ├─assets         （资源目录）
    |  ├─components     （系统组件比如编辑区呀，物料组件区、配置区）
    |  ├─custom-components         （物料组件目录）
    |  ├─router         （路由）
    |  ├─store          （数据存储）
    |  ├─utils          （工具包）
    |  ├─views          （路由视图层）
    |  ├─App.vue        （app组件）
    |  ├─main.js        （入口文件）
    ├─public            （包含模板文件）
    ├─.gitigonre        （git的忽略配置文件）
    ├─babel.config.js  	（babel的配置文件，babel是js编译器）
    ├─package.json  	（npm包管理的配置文件）
    ├─vue.config.js     （vue的配置文件）

```
